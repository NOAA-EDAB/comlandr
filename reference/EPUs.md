# EPUs: Properties of Ecological Production Units (EPUs)

Lists the four EPUs (Georges Bank (GB), Gulf of Maine (GOM), Mid
Atlantic Bight (MAB), Scotian Shelf (SS)) with the statistical areas
that make up each region and their area (km^2)

## Usage

``` r
EPUs
```

## Format

A list of lists

- GB:

  Georges Bank EPU

- GOM:

  Gulf of Maine EPU

- MAB:

  Mid Atlantic Bight EPU

- SS:

  Scotian Shelf EPU

## List components

Each list item is a list itself containg the following elements:

- statAreas:

  statistical areas that comprise the EPU

- area:

  area in km^2 of toal EPU
