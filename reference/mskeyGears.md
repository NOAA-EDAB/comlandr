# mskeyGears: Gear Fleet mapping from mskeyrun project

Associates NEGEAR2 codes to Fleets

## Usage

``` r
mskeyGears
```

## Format

A Data frame

- NEGEAR2:

  NEFSC 2 digit Gear code

- MESHCAT:

  Category of mesh size, SM, LG

- Fleet:

  Name of Fleet
