# mskeyAreas: Area Species, EPU designations

Option to pass to `get_comland_data` if the user wishes define
proportion Statistical Areas based on findings in mskeyrun project.
Statistical Area that straddle EPUs have landings proportioned among
neighboring EPUs

## Usage

``` r
mskeyAreas
```

## Format

A Data frame

- AREA:

  Statistical Area

- NESPP3:

  Gulf of Maine EPU

- MeanProp:

  Proportion of Statistical area to attribute to EPU

- EPU:

  Ecological Production Unit
